# Automotive Telemetry Diagnostic & Predictive Maintenance Toolkit

## Overview

A Python-based engineering toolkit developed to analyze vehicle telemetry datasets, validate sensor readings, detect anomalies, and generate diagnostic reports for predictive maintenance and system debugging.

## Features

* Import vehicle telemetry datasets (CSV)
* Data validation and cleaning
* Missing value detection
* Duplicate record detection
* Engine temperature analysis
* RPM and speed analysis
* Battery voltage monitoring
* Fault detection
* Diagnostic report generation
* Data visualization

## Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib

## Project Workflow

1. Import telemetry dataset.
2. Validate and clean data.
3. Detect abnormal sensor values.
4. Generate visualizations.
5. Produce a diagnostic report.
